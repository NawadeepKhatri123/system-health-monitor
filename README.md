# System Health Monitor

A lightweight Python-based system monitoring tool designed to track basic system performance metrics and support foundational system administration and troubleshooting workflows.

---

## Features

- Real-time monitoring of:
  - CPU usage
  - Memory (RAM) usage
  - Disk usage
  - Battery status (if applicable)

- Configurable warning thresholds for system resource usage  
- Continuous polling at set intervals  
- Logging of system performance metrics to file  
- Lightweight and cross-platform compatible (Linux-focused)

---

## Purpose

This project was built to practice system monitoring concepts commonly used in IT support and infrastructure environments, including:

- System performance tracking  
- Log generation and analysis  
- Resource usage monitoring  
- Foundational concepts used in enterprise monitoring tools (e.g., NOC dashboards)
